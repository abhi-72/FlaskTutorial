# FlaskTutorial

## Prerequisites
  + Python3 or higher
  + pip3 or higher

## Steps to run
  + Get inside the folder
    > cd FlaskTutorial
  + Create a python virtual environment
    > python3 -m virtualenv env_flasktutorial
  + Install required python packages
    > pip3 install -r requirements.txt
  + Run the application
    > python3 app.py
